# 🤖 AI Agents vs Chatbots (සාමාන්‍ය Chatbot සහ AI Agent අතර වෙනස)

AI තාක්ෂණයේ පරිණාමයත් සමඟ දැන් අපි Generative AI (පෙළ හෝ රූප නිර්මාණය කිරීම) යුගයෙන් **Agentic AI** (ස්වාධීනව වැඩ කළ හැකි පද්ධති) යුගයට ගමන් කරමින් සිටිනවා. 

ගොඩක් අය Chatbot කෙනෙක් සහ AI Agent කෙනෙක් පටලවා ගන්නවා. නමුත් මේ දෙක අතර තාක්ෂණිකව ඉතාමත් පැහැදිලි වෙනසක් තියෙනවා.

---

## 💬 1. සාමාන්‍ය Chatbot කෙනෙක් කියන්නේ මොකක්ද? (Conversational AI)

Chatbot කෙනෙක් කියන්නේ ප්‍රධාන වශයෙන්ම User සමඟ සංවාදයේ යෙදීමට (Conversation) නිර්මාණය කර ඇති පද්ධතියක්.
* **ක්‍රියාකාරීත්වය:** User ප්‍රශ්නයක් ඇසූ විට, Chatbot එක සතු දැනුම පාවිච්චි කරලා ඒකට උත්තරයක් දෙනවා.
* **සීමාවන්:** උන්ට තනියම තීරණ ගන්න බැහැ. වෙනත් බාහිර Tools (උදා: File එකක් සේව් කිරීම, Email එකක් යැවීම) පාවිච්චි කරන්න බැහැ. හැමතිස්සෙම ඊළඟට කරන්න ඕන දේ User විසින්ම කියන්න ඕනේ.

> **💡 උදාහරණයක්:** ඔයා Chatbot කෙනෙකුට "මට ලබන සතියේ Trip එකක් යන්න Plan එකක් හදලා දෙන්න" කිව්වොත්, එයා ලස්සන සැලසුමක් ලියලා දෙවි. හැමතිස්සෙම වැඩේ නතර වෙන්නේ "Text" එකකින් පමණයි.

---

## 🚀 2. AI Agent කෙනෙක් කියන්නේ මොකක්ද? (Autonomous Systems)

AI Agent කෙනෙක් කියන්නේ හුදෙක් කතා කරනවාට වඩා, දී ඇති ඉලක්කයක් (Goal) සපුරා ගැනීම සඳහා **ස්වාධීනව (Autonomous) සැලසුම් සකස් කරමින්, බාහිර මෙවලම් (Tools) භාවිතා කරමින් ක්‍රියා කළ හැකි** උසස් පද්ධතියක්.

AI Agent කෙනෙකුට ප්‍රධාන කොටස් 4ක් තියෙනවා:
1. **Core LLM (මොළය):** තීරණ ගැනීමට සහ තර්ක කිරීමට පාවිච්චි කරයි.
2. **Planning (සැලසුම්කරණය):** ලොකු කාර්යයක් පොඩි කොටස් වලට කඩාගෙන කරන ආකාරය ප්ලෑන් කරයි (Self-reflection / Chain of Thought).
3. **Memory (මතකය):** කෙටි කාලීන සහ දිගු කාලීන මතකය තබා ගනිමින් පෙර සිදුවූ දේ මතක තබා ගනියි.
4. **Tools (මෙවලම් භාවිතය):** Web search කිරීම, Calculators පාවිච්චි කිරීම, APIs රන් කිරීම, Database එකකට Data ලිවීම වැනි බාහිර වැඩ තනිවම සිදු කරයි.

> **💡 උදාහරණයක්:** ඔයා AI Agent කෙනෙකුට "මට ලබන සතියේ Trip එකක් යන්න Plan එකක් හදලා, ඒකට හරියන Hotel එකක් බුක් කරලා, මගේ Calendar එකට ඒක ඇතුළත් කරන්න" කිව්වොත්; Agent විසින්ම Web Search කරලා හොඳම Hotel එක හොයාගෙන, Hotel API එක හරහා ඒක Book කරලා, ඔයාගේ Google Calendar එකට Event එකකුත් දේවි!

## 📊 Chatbot සහ AI Agent අතර ප්‍රධාන වෙනස්කම්



| ලක්ෂණය | 💬 සාමාන්‍ය Chatbot | 🤖 AI Agent |
| :--- | :--- | :--- |
| **ප්‍රධාන කාර්යය** | ප්‍රශ්න වලට පිළිතුරු දීම (Text Output). | දී ඇති ඉලක්කයක් සම්පූර්ණ කිරීම (Action Input/Output). |
| **ස්වාධීනත්වය** | ඉතා අඩුයි (User හැම පියවරක්ම කිව යුතුයි). | ඉතා වැඩියි (තනිවම පියවරෙන් පියවර සැලසුම් කරයි). |
| **Tools භාවිතය** | බාහිර පරිගණක මෙවලම් හෝ APIs භාවිතා කළ නොහැක. | APIs, Web Browsers, සහ Databases පාවිච්චි කළ හැක. |
| **භාවිතය (Example)** | Customer Support, Q&A Dashboards. | Auto-coding agents, AI Web Scrapers, Autonomous Personal Assistants. |
